OPTION 2 - Manual JavaScript:
─────────────────────────────────────────────────────────────────────────────
Replace the hardcoded md object with:
  let md = {};
  fetch('menu_data.json')
    .then(res => res.json())
    .then(data => {
      md = data;
      renderMenu();
    });

═══════════════════════════════════════════════════════════════════════════════
  🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

PROBLEM: Can't access admin panel
SOLUTION: 
  - Check XAMPP Apache is running
  - Verify URL: http://localhost/ZAD/admin.php
  - Clear browser cache

PROBLEM: "Unauthorized" error
SOLUTION:
  - Make sure you're logged in
  - Check PHP sessions are enabled
  - Try logging out and back in

PROBLEM: Images not uploading
SOLUTION:
  - Check /images/ folder permissions (must be writable)
  - Verify file size < 5MB
  - Ensure file type is JPG, PNG, WebP, or GIF

PROBLEM: Changes not saving
SOLUTION:
  - Check file permissions on JSON files
  - Look for PHP errors in browser console (F12)
  - Check XAMPP error logs
